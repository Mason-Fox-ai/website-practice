## 🧑‍💻 Диаграмма прецедентов (Use Case)

![Кто пользуется сайтом](Кто%20пользуется%20сайтом.drawio.png)

- **Гость** → просматривает страницы, читает статьи  
- **Пользователь** → дополнительно оставляет комментарии  
- **Администратор** → добавляет/удаляет статьи и управляет пользователями

---

## ☁️ Диаграмма развёртывания (Deployment)

![Диаграмма развёртывания](Диаграмма%20развёртывания.drawio.png)

- **Браузер** (компьютер пользователя) ↔ **Сервер** (Apache, PHP, MySQL) через **HTTP**  
- **Сервер** ↔ **GitHub** (репозиторий кода) через **Git**
